Eases

Predefined easing functions that control velocity over time. GSAP includes Power, Expo, Circ, Back, Elastic, Bounce, Steps, and more — each with in, out, and inOut variants.

Springs

Spring approximations using GSAP's Elastic ease. amplitude maps to stiffness; period maps to damping. Each entry notes the CSS spring() equivalent for cross-reference.

Staggers

Offset each element's animation start time. Control by total amount, per-element each duration, origin from point, or a grid layout for 2D spatial offsets.

Playback Controls

Repeat, yoyo, delay, timeline positioning, and time scale. The mechanics of controlling when and how animations play — independent of the ease applied.
